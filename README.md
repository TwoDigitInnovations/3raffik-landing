# 3raffik Landing Page

A modern, professional landing page for **3raffik** - an innovative affiliate marketing platform that connects companies with affiliates to promote products through referral links and QR codes.

## 🚀 Features

- **Modern Design**: Clean, professional aesthetic with 3raffik brand colors
- **Fully Responsive**: Mobile-first design that works on all devices
- **Interactive Components**: Smooth animations and hover effects
- **SEO Optimized**: Proper meta tags and structured content
- **Accessibility Compliant**: WCAG guidelines followed
- **Performance Optimized**: Fast loading with optimized components

## 🎨 Brand Colors

- **Primary Yellow**: #FFCC00
- **Accent Red**: #EF0027
- **Black**: #000000
- **White**: #FFFFFF
- **Gray**: #666666

## 📱 Sections Included

1. **Header/Navigation** - Responsive navbar with mobile menu
2. **Hero Section** - Compelling headline with visual illustration
3. **How It Works** - Step-by-step process for companies and affiliates
4. **Features** - Detailed feature breakdown for both user types
5. **Benefits** - Key value propositions with statistics
6. **App Showcase** - Mobile app promotion with mockups
7. **Testimonials** - Customer success stories and reviews
8. **Pricing** - Three-tier pricing structure
9. **FAQ** - Common questions and answers
10. **Footer** - Contact info, links, and app downloads

## 🛠️ Tech Stack

- **Next.js** - React framework
- **Tailwind CSS** - Utility-first CSS framework
- **Lucide React** - Beautiful icons
- **Inter Font** - Modern typography

## 🚀 Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

3. **Open your browser:**
   Navigate to [http://localhost:3000](http://localhost:3000)
